# 别回头
我们就喜欢在对答案的时候责怪当初的自己，

总忍不住想要是当初选了另一个答案就能多拿几分，

结果就全然不同。

可当时的自己困在屋里本就满心迷茫，

如今的你回头审判过去的你，

何尝不是一种对自己的霸凌呢。


不要去美化那条你没选的路，

人生没有白走的路，

对错都算数。

你脚下的这条路是最好的路，

终会有星辰大海，

花团锦簇。

眼睛藏在前面，

永远别向后看。

情出自愿，

事过无悔。

得之坦然，

失之淡然。


选错了就选错了，

不要一遍一遍的后悔，

也不要总是责怪从前的自己，

因为他当时站在雾里也很迷茫。

所以不必回头看。


蔷薇今年谢了，

明年还会再开，

总会有人陪你看不同的风景。

勇敢点，

向前走，

别回头。
